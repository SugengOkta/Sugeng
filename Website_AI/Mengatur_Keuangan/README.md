# Personal Finance Tracker

A simple and intuitive web application to manage your personal finances. Track your income, expenses, and savings with ease.

## Features

*   **Transaction Tracking:** Record your income, expenses, and savings.
*   **Budget Management:** Set monthly budgets for different expense categories.
*   **Savings Goals:** Define and track your progress towards your financial goals.
*   **Recurring Transactions:** Automate the entry of regular transactions.
*   **Financial Insights:** Visualize your financial trends and expense breakdowns with charts.
*   **User Authentication:** Secure your financial data with a personal account.
*   **Profile Customization:** Personalize your profile with a profile picture.

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

*   [Node.js](https://nodejs.org/)
*   [Firebase CLI](https://firebase.google.com/docs/cli)

### Installation

1.  Clone the repo
    ```sh
    git clone https://github.com/your_username_/your_project_name.git
    ```
2.  Install NPM packages
    ```sh
    npm install
    ```

### Running Locally

1.  Serve the project using Firebase CLI:
    ```sh
    firebase serve
    ```
2.  Open your browser and navigate to `http://localhost:5000`.

## Deployment

To deploy the project to Firebase Hosting, run the following command:

```sh
firebase deploy
```
